Scrieti un program pentru vizualizarea algoritmilor de sortare: un vector
este reprezentat printr-un sir de bile etichetate pe ecran si este sortat,
fiecare interschimbare fiind vizualizata prin deplasarea bilelor corespunzatoare
(animatie). Daca vectorul este prea mare se face automat scroll orizontal a.i.
bila care se deplaseaza sa fie mereu pe ecran.
Meniu cu: editarea, incarcarea/salvarea in fisier a vectorului, alegera
algoritmului de sortare (cel putin 5 algoritmi).
Se va folosi biblioteca Xlib.
